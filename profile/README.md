# FINANZTRACKER

Finaztracker ist eine GitHub-Organisation für die Entwicklung eines Finanzverwaltungssystems im Rahmen eines Schulprojekts.

Unser Ziel ist die Entwicklung eines einfachen Programms zur Verwaltung von Einnahmen und Ausgaben. Dabei arbeiten wir gemeinsam an der Planung, Umsetzung und Weiterentwicklung der Software.

## 📦 PROJEKT

Aktuelles Hauptprojekt:

- **Finanztracker** – eine Anwendung zur Erfassung, Speicherung und Auswertung persönlicher Finanzdaten.

## 🛠 TECHNOLOGIEN

- Java
- mögliche GUI mit JavaFX
- relationale Datenbanken (MySQL)
- Git & GitHub zur Versionsverwaltung

## 🤝 ZUSAMMENARBEIT ZWISCHEN UNS

Die Entwicklung erfolgt im Team über GitHub.  
Jedes Teammitglied arbeitet an einzelnen Komponenten und trägt zur Weiterentwicklung des Projekts bei.
